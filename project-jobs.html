Data Analyst Job Market Analysis
Descriptive & Prescriptive Analytics · Labour Market Domain · In Progress
Overview

A mixed descriptive and prescriptive analysis of the current data analyst job market — identifying skill demand patterns, salary distributions by region and experience level, and producing concrete, evidence-grounded recommendations for early-career analysts on where to focus their skill development.

Unlike the FIFA project which is purely descriptive, this project adds a prescriptive layer: specific, actionable recommendations backed by the descriptive findings. This distinction between what the data shows and what should be done about it is documented explicitly throughout.

Approach
  • Python-primary analytical pipeline with SQL for aggregation and Tableau for the visualisation layer
  • Excel stakeholder deliverables alongside interactive Tableau dashboards
  • Prescriptive layer grounds recommendations in the descriptive findings — no recommendation is made without an explicit evidential basis
  • Scope boundaries documented: salary recommendations are regional and role-specific; skill demand patterns are platform and date-specific
Status

This project is currently in progress. Repository and full write-up will be available on completion. The Tableau dashboard will be the primary visual deliverable.
